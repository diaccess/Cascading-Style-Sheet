Belajar CSS Layouting Display

Inline
elemen HTML secara default tidak menambahkan garis baru
lebar dan tingginya selebar kontekn yang ada didalamnya
tidak bisa mengatur tinggi dan lebar dari konten
margin dan padding hanya mempengaruhi elemen secara horizontal, tidak vertikal

Inline-blcok
tidak ada element secara default memiliki properti Display: Inline-blcok
user harus merubah secara manual properti tersebut
prilaku dasarnya sama seperti Inline
perbedaanya, element Inline-blcok dapat diatur tinggi dan lebarnya.

block
secara default menambahkan garis baru ketika element dibuat
jika lebarnya tidak di atur, maka lebar defaultnya akan memenuhi window/parrent
dapat mengatur lebar dan tingginya
didalam element block kita dapat menyimpan tag dengan element inline, Inline-blcok, atau block
